## About

Vioneta Agro allows anyone to create add-on repositories to share their
add-ons for Vioneta Agro easily. This repository is one of those repositories,
providing extra Vioneta Agro add-ons for your installation.

The primary goal of this project is to provide you (as a Vioneta Agro user)
with additional, high quality, add-ons that allow you to take your automations to the next level.

## Installation

In general, there is no need to install this repository on your
Vioneta Agro instance. It is activated and added by Vioneta Agro
by default.

However, if the repository is missing on your setup, adding this add-ons
repository to your Vioneta Agro instance is pretty easy. In the
Vioneta Agro add-on store, a possibility to add a repository is provided.

Use the following URL to add this repository:

```txt
https://github.com/Vioneta/repository
```

## Releases

Releases are based on [Semantic Versioning][semver], and use the format
of `MAJOR.MINOR.PATCH`. In a nutshell, the version will be incremented
based on the following:

- `MAJOR`: Incompatible or major changes.
- `MINOR`: Backwards-compatible new features and enhancements.
- `PATCH`: Backwards-compatible bugfixes and package updates.
